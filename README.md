# daniel-martins-portfolio

Meu portfólio pessoal com projetos e informações de contato.

**[→ Acessar o site](https://daniel-martins-portfolio.vercel.app)**

---

## Stack

- React 19
- TypeScript
- Vite
- Vercel

## Rodando localmente

```bash
git clone https://github.com/danielrmartins/portfolio
cd portfolio
npm install
npm run dev
```

Acesse `http://localhost:5173`

## Deploy

```bash
vercel --prod
```
